Base binaria vs base decimal

¿Cómo contamos en base binaria?

Aquí podrás observar cómo se cuenta en base binaria y comparar con la base decimal que ya conoces. En particular, nota cada cuando en la base binaria es necesario agregar un dígito para poder seguir contando.

Podrás agregar elementos a contar, y también podrás reiniciar la escena para empezar desde cero.

El número de elementos contados se presenta como líneas y taches.

Presta atención al conteo, que se hace del dígito de la derecha y hacia la izquierda. Observa cómo cambian los símbolos en cada dígito y qué ocurre cuando te has gastado todos los símbolos tanto en el caso binario como en el decimal.

Una vez que hayas revisado el interactivo, intenta encontrar una fórmula que te permita saber cuántos palitos puedes contar usando un cierto número de dígitos en binario ¿Cuál sería la contraparte de dicha fórmula para el caso decimal?

A continuación usarás un interactivo para ver cómo se cuenta con una base binaria. Revisa bien las instrucciones al principio del interactivo.

Fórmula para saber cuántos palitos se pueden contar

Ahora que ya has practicado hasta contar bastantes palitos en base binaria, habrás notado que cada dígito cuenta con 2 símbolos (el 0 y el 1). Por cada dígito que se agrega, se multiplica el número de opciones pasadas por 2 (el número de símbolos).

Así pues, si tienes 3 dígitos, el número de opciones de conteo que tienes es \(2)(2)(2)=2^3=8\). De forma general, si tienes \(n\) dígitos, puedes contar \(2^n\) palitos en binario.

Observa que algo parecido pasa cuando usas base decimal. ¿Se te ocurre una fórmula para cuántos palitos puedes contar dado un número de dígitos n en la base decimal?
